# 4_6_mesh

地図上で固定サイズのメッシュを生成し、  
**GeoJSON** と **PNG** として出力できる Web アプリです。

OpenLayers を使用しており、GitHub Pages 上で動作します。

デモ  
https://0712kazu.github.io/4_6_mesh/

---

# 概要

地図上をクリックすると、その地点を基準に **指定サイズのメッシュグリッド**を生成します。

メッシュは **EPSG:3857（Webメルカトル）座標系上で正方形**として生成されます。

主な用途

- GIS用メッシュ作成
- 調査区画作成
- 図面作成用グリッド
- メルカトル座標ベースのグリッド生成

---

# 主な機能

## メッシュ生成

- 地図クリックでメッシュ作成
- 横マス数・縦マス数を指定可能
- セルサイズを指定可能
- メッシュは **EPSG:3857（Webメルカトル）上で正方形**

## 中心座標表示

クリックした地点の座標を表示

表示座標

- EPSG:3857（m）
- WGS84（緯度経度）

---

# メッシュサイズ

UI の「1マスサイズ」は  
**Webメルカトル座標系（EPSG:3857）上のセルサイズ（m）** を意味します。

例

```
1500
```

の場合

- メルカトル座標上  
  - 東西：1500 m  
  - 南北：1500 m  

ただし Web メルカトルの特性上、  
**実際の地表距離としての南北方向は緯度により変化します。**

|方向|距離|
|---|---|
|東西|約1500 m|
|南北|緯度により変化|

このツールでは  
**投影座標系（EPSG:3857）上の正方形メッシュ生成**を優先しています。

---

# メッシュ生成の安定化処理

クリックした座標が **メッシュ境界と一致する場合の不安定さ**を避けるため、  
メッシュ生成時のみ内部的に以下の処理を行っています。

**中心座標を北東方向へ 1/4 メッシュ移動**

```
mesh_center_x = click_x + cellSize / 4
mesh_center_y = click_y + cellSize / 4
```

用途ごとの座標の扱い

|用途|使用座標|
|---|---|
|表示座標|クリック位置|
|ファイル名|クリック位置|
|GeoJSON属性 center_x / center_y|クリック位置|
|メッシュ生成|1/4メッシュオフセット|

この処理により

- メッシュ境界一致
- 浮動小数誤差
- タイル境界問題

などを回避しています。

# 背景地図

地理院タイルを使用しています。

淡色地図  
https://cyberjapandata.gsi.go.jp/xyz/pale/{z}/{x}/{y}.png

陰影起伏図  
https://cyberjapandata.gsi.go.jp/xyz/hillshademap/{z}/{x}/{y}.png

表示方式

```
淡色地図
×
陰影起伏図（乗算）
```

背景透過率は UI から変更できます。  
デフォルトは **55%** です。

# 出力機能

## GeoJSON

座標系

```
EPSG:4326（緯度経度）
```

各ポリゴン属性

```
row
col
center_x
center_y
mesh_center_x
mesh_center_y
cell_size
```

---

## PNG

PNG出力内容

- 背景地図
- 陰影起伏図
- メッシュ

PNG は **メッシュ範囲に自動ズームして書き出します。**

---

# ファイル名

デフォルト

```
mesh_{centerX}_{centerY}
```

例

```
mesh_15544394.472_4230071.807
```

UIから変更できます。

# 使い方

1. メッシュ設定を指定  
   - セルサイズ  
   - 横マス数  
   - 縦マス数  

2. 地図をクリック

3. メッシュ生成

4. 必要に応じて

- 背景透過率調整
- ファイル名変更

5. 出力

```
GeoJSONを書き出し
PNGを書き出し
```
